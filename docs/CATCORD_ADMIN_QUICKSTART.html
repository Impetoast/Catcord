Catcord – Admin Quickstart

One-page setup and operations checklist for Discord server administrators.

1) Permissions

  • Send Messages, Manage Webhooks, Read Message History
  • If using threads: Create Public Threads
  • Optional diagnostics: allow Use Application Commands for staff running slash commands.

2) Map Channels

/langrelay_group_create EU
/langrelay_group_add EU #german DE
/langrelay_group_add EU #english EN-US
/langrelay_group_add EU #french FR
/langrelay_group_list

Use autocomplete for groups and language codes; remove entries with /langrelay_group_remove.

3) Choose Provider

/langrelay_provider provider:deepl
# or:
/langrelay_provider provider:openai

Bot validates tokens before switching. Default provider favours OpenAI when both keys exist.

4) Toggle Options

/langrelay_replymode state:on
/langrelay_thread_mirroring state:on
/langrelay_reaction_mirroring state:on

Leave options off if you do not need reply context, thread mirroring, or reaction sync.

5) Power Controls

/langrelay_power state:on
/langrelay_group_power group:EU state:on
/langrelay_status

Use group power to pause a single mapping while keeping others online.

6) Auto-Translate (Optional)

/autotranslate_on target:EN-US source:DE formality:more min_chars:8
/autotranslate_status
/autotranslate_off

Enable in channels without relay mappings. Keep min_chars low enough for natural replies.

7) Schedule Reminders

/reminder add name:backup channel:#ops message:"Run backup" time:02:00
/reminder add name:reports channel:#ops message:"Weekly metrics" times:"Mon@09:00, Thu@09:00" group:Ops
/reminder list
/reminder toggle enabled:false

Use /reminder edit for adjustments and /reminder group rename/... remove to manage labelled sets.

Troubleshooting

  • Nothing mirrored → check permissions, group power, and /langrelay_status.
  • Translation fails → verify provider tokens and API quotas.
  • Threads not mirrored → ensure thread option and permissions are active.
  • Reactions not mirrored → enable reaction mirroring on both ends.

Useful Utilities

/translate  text:<message>
/detect     text:<message>
/languages
/tping
/ping
/about

Slash commands answer ephemerally by default. Use /hello for a quick connectivity ping.

Tip: Keep consistent channel names (e.g., #german, #english, #french) so you can re-use autocomplete and avoid mapping duplicates.

Catcord – Admin Quickstart

One-page setup and operations checklist for Discord server administrators.

1) Permissions

  • Send Messages, Manage Webhooks, Read Message History
  • If using threads: Create Public Threads

2) Map Channels

/langrelay_group_create EU
/langrelay_group_add EU #german DE
/langrelay_group_add EU #english EN
/langrelay_group_add EU #french FR
/langrelay_group_list

3) Choose Provider

/langrelay_provider provider:deepl
# or:
/langrelay_provider provider:openai

4) Toggle Options

/langrelay_replymode state:on
/langrelay_thread_mirroring state:on
/langrelay_reaction_mirroring state:on

5) Verify

/langrelay_status

6) Power (Global On/Off)

/langrelay_power state:off
# later
/langrelay_power state:on

Group Power

/langrelay_group_power group:EU state:off
# later
/langrelay_group_power group:EU state:on

7) Auto-Translate (Optional)

/autotranslate_on target:EN
/autotranslate_status
/autotranslate_off

8) Schedule Reminders

/reminder add name:backup channel:#general message:"Run backup" time:02:00
/reminder list
/reminder remove name:backup

Troubleshooting

  • Nothing mirrored → check mappings & permissions.
  • Translation fails → verify keys / provider.
  • Threads not mirrored → ensure thread permission.
  • Reactions not mirrored → ensure reaction mirroring is enabled.

Useful Utilities

/translate  text:<message>
/detect     text:<message>
/languages
/ping
/about
Tip: Use consistent channel names (e.g., #german, #english, #french) and map each once.

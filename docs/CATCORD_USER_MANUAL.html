Catcord – User Manual

How to use Catcord as a regular Discord member.

Overview

Catcord helps people chat across languages. When you write in a language-specific channel (for example, #german), Catcord can mirror your message to other mapped channels (such as #english or #french). Mirrored messages may be automatically translated by DeepL or OpenAI and are posted via webhooks using your display name and avatar—so they look like you sent them.

How It Works

  • Channel mappings: Admins connect language channels (e.g., #german ↔ #english ↔ #french). Each group can be paused independently.
  • Invisible posting: Catcord uses webhooks to post mirrored messages with your name and avatar.
  • Translation: Depending on server settings, messages are translated automatically (DeepL or OpenAI).
  • Auto-translate channels: Some channels may have /autotranslate_on enabled—messages get an instant reply containing the translation.
  • Loop protection: Messages posted by webhooks aren’t mirrored again.
  • Extras: Optional reply snippets, thread mirroring, and mirrored emoji reactions can be enabled by admins.

Using Catcord

Post messages

  • Write normally in your language channel. If channels are mapped, your message will appear in the other language channels.
  • Attachments (images/files) are mirrored with your message when possible.

Auto-translated replies

  • In channels with auto-translate, expect a bot reply beneath your message summarising the translation.
  • Keep your text longer than the minimum character threshold (typically 5–10 characters) so translations trigger.

Replies

  • If enabled by admins, mirrored replies include a short translated line indicating what you replied to.

Threads

  • If enabled by admins, messages in a thread are mirrored into threads with the same name in the other channels.

Reactions

  • Emoji reactions you add to mirrored messages can be copied to the linked copies when reaction mirroring is on.

Tips

  • Keep sentences clear and simple for better translations.
  • If a translation seems off, try rephrasing and sending again.
  • Mentions may be sanitised to avoid pinging people across different channels.
  • Thread names should match across languages to keep mirrored discussions tidy.

Slash Commands You Can Use

  • /translate – translate a custom snippet (results arrive privately by default).
  • /detect – detect the language of a message.
  • /languages – list supported DeepL language codes.
  • /ping – check the bot’s latency.
  • /about – see bot information.

Admins may grant additional commands such as /reminder add for scheduling messages. If unsure, ask your moderation team.

Reminders

  • Servers can schedule reminders with Catcord. When a reminder is configured for your channel, it will appear on the minute (UTC) with the text provided by admins.
  • Reminders may include an embed headline or appear as plain text.
  • If the channel participates in a language relay group, the reminder can be translated and echoed to the paired channels automatically.

Privacy & Transparency

  • Mirrored posts use your display name and avatar via webhooks.
  • Catcord does not DM you and does not store your personal messages beyond what is needed to mirror them.
  • Admins control which channels are mapped and which features are enabled.

Troubleshooting

  • My message isn’t mirrored: Ask an admin whether your channel is mapped and whether you have permission to post in the target channels.
  • Attachment didn’t appear: Retry with a smaller file; very large attachments can fail.
  • Translation looks wrong: Automatic translation isn’t perfect. Rephrase and resend, or ask an admin which provider is used.
  • I see duplicated reminders: The server might run both relays and reminders; let admins know so they can adjust settings.
